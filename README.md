# Web Foundations

This repository documents my learning journey through modern web development.
The focus is on building strong fundamentals before moving to advanced tools
and full-stack development.

## Topics
- HTML & CSS
- Tailwind CSS
- SASS / SCSS
- JavaScript (ES6+)
- React
- Node.js
- Next.js

## Structure
Each folder represents a learning section and includes its own README
with notes, exercises, and mini-projects.

## Progress
Code is added and updated on a near-daily basis as part of continuous learning.

## Goal
To develop solid web foundations and progress toward building modern,
scalable web applications.
