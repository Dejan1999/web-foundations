Selector Types:

  1. Element Selector ( h3 { color: blue; } )
  2. Class Selector ( .some-class { color: red; } )
  3. Universal Selector ( * { background-color: green; } )
  4. ID Selector ( #some-id { color: yellow; } )
  5. Attribute Selector ( [disabled] { color: purple;} )
  1. Element Selector (h1, p, img) and pseudo-elements (::after, ::before)
  2. Class Selector (.some-class) and pseudo-class (:pseudo-class) and Attribute Selector([disabled])
  3. ID Selector (#some-id)
  4. Inline CSS Style (h1 color: red;) Not Recommended
